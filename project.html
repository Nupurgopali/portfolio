Our main aim was to create an app that helps the user navigate inside a building,mall or any such structure, without using internet.Since it is very difficult to have internet connection during emergencies.

SolarMate


The main idea was to measure the current and voltage parameters of the solar panels set up in establishments using solar energy for power generation, and also to collect data about the factors affecting the amount of power generated, like temperature, humidity, cloud coverage, and insolation. Also, the solar panel will rotate according to the direction of incoming sunlight.

Non-Contact Water Level Monitoring System using Arduino and Ultrasonic Sensor


The main aim was to design a Non-Contact Water Level Monitoring Control System. In this project the Ultrasonic sensor provides the feedback to system, this feedback is then used by the system to decide whether the pump will be operated or not. The Ultrasonic Sensor obtains the water level as distance of water surface from the sensor.

Poultry Farm- Using Arduino Uno board


This project aims to provide an ambient atmosphere for poultry animals. The project uses basic sensors such as DHT11, MQ135 to provide ambient temperature for the hens to breed and to even send an alert if the level of CO2 increases above an optimum level.Like wise other factors that are essential to provide healthy environment for poultry animals are also monitored.

Traffic light controller - Using FPGA board


To make a traffic light controller that controls the traffic in a cross road based on the number of vehicles in each road. The idea is that road with a greater number of vehicles gets the priority, this way the road with the highest traffic will be cleared first hence making the driving experience smoother
